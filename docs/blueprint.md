# **App Name**: Keymara

## Core Features:

- Hero Section Display: Display a captivating hero section with a clear value proposition, similar to the Keymara website.
- Category Navigation: Provide a clear and intuitive navigation menu with links to all ad categories, mirroring the style of the Keymara website.
- Category Pages: Create dedicated pages for each ad category, showcasing relevant content in a layout consistent with the Keymara website.
- AI-Powered Ad Insights Tool: Offer a tool that uses AI to analyze ad performance and suggest improvements.

## Style Guidelines:

- Primary color: Soft yellow (#FFDA63) for a warm and inviting feel, as seen on the Keymara website.
- Background color: Very light beige (#F5F5DC), providing a clean and airy backdrop reminiscent of the Keymara website.
- Accent color: Bright yellow (#FFB347) to draw attention to calls to action, matching the Keymara brand.
- Body font: A clean and modern sans-serif font like 'Open Sans' for readability and a user-friendly experience.
- Headline font: A bold and impactful font such as 'Montserrat' to create visually appealing headers.
- Use simple, clear icons with a touch of yellow to represent each ad category, aligning with the Keymara aesthetic.
- Maintain a clean and organized layout with generous whitespace, similar to the Keymara website, for a professional and uncluttered feel.
